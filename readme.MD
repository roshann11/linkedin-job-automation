# LinkedIn Job Automation

## Features

1. Login to LinkedIn
2. Search posts using keywords
3. Extract recruiter emails
4. Send application email
5. Attach resume
6. Generate CSV report

## Place your resume in the /resume folder

## .env
LINKEDIN_EMAIL
LINKEDIN_PASSWORD
GMAIL_USER
GMAIL_APP_PASSWORD
GEMINI_API_KEY

## Run

npm install

node app.js
